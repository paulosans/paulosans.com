# paulosans.com

Personal portfolio of Paulo Santos — staff designer at Mercado Livre with 12+ years of experience. A curated space for case studies, design thinking, and product craft.

## Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Agrandir** — custom typeface

## Features

- Animated menu with per-page backgrounds and crossfade transitions
- Dark / light mode toggle with sun ↔ moon animation
- Fully responsive — desktop, tablet, and mobile
- Interactive hover states faithful to Figma prototypes

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design

Designed in Figma. All interactions and motion are derived directly from the original prototypes.
